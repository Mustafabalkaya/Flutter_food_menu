# Flutter Food Menu App

This Flutter project is designed to create a food menu application. The app includes features such as displaying categories of dishes, listing meals within a chosen category, viewing detailed information about a meal, and adding items to a shopping cart.

## Features

- **Home Screen:** Display all food categories.
- **Category Details:** List meals within the selected category.
- **Meal Details:** View details of a selected meal (price, ingredients, etc.).
- **Cart:** Add meals to the cart and view cart contents.

## Technologies Used

- **Flutter:** Framework used for UI design and application Logic.
- **Dart:** Programming language used for the backend logic of the Flutter app.
- **Firebase:** Cloud-based solution used for user authentication and storing cart information.

## Installation

1. Clone this repository to your machine.
   ```bash
   git clone https://github.com/Mustafabalkaya/flutter-food-menu.git
   ```

2. Navigate to the project directory.
   ```bash
   cd flutter-food-menu
   ```

3. Install the necessary dependencies.
   ```bash
   flutter pub get
   ```

4. Create a Firebase project and add your Firebase configuration information.

5. Run the application.
   ```bash
   flutter run
   ```
